# Registro de Edades

Aplicación web para recopilar y analizar datos demográficos mediante el ingreso de edades de usuarios.  
Incluye validación de rango (1–120 años), barra de progreso y visualización dinámica de porcentajes por grupo poblacional.  
Genera un análisis automático con texto interpretativo y métricas de confiabilidad y sesgo.  
Diseñada con HTML, CSS y JavaScript, utilizando variables globales y componentes reutilizables para un dashboard responsivo.  
Ideal como práctica de frontend y ejemplo de visualización de datos en tiempo real.
